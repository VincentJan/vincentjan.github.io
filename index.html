第七章 微分方程（二）

一阶微分方程的种类：

  1. 可分离变量的微分方程
  2. 齐次微分方程
  3. 一阶齐次线性微分方程
  4. 一阶非齐线性微分方程
  5. Bernoulli方程
  6. 全微分方程

一阶微分方程的解法：

  1. 可分离变量的微分方程：

    • 形式：
      $$\frac{dy}{dx}=f(x,y),\quad f(x,y) = \varphi_1(x)\varphi_2(y)$$
      • 解法：
        $$\frac{dy}{dx} = f(x,y) \implies \frac{dy}{dx} = \varphi_1(x)\varphi_2(y) \implies \frac{dy}{\varphi_2(y)} = \frac{\varphi_1(x)}{dx} \ \implies \int\frac{dy}{\varphi_2(y)} = \int\frac{\varphi_1(x)}{dx} + C$$

  2. 齐次微分方程:

    • 形式：
      $$\frac{dy}{dx} = f(x,y), f(x,y)=\varphi(\frac{y}{x}) \lor f(tx,ty) = f(x,y)$$
    • 解法：
      $$\frac{dy}{dx} = f(x,y) \implies \frac{dy}{dx} = f(\frac{y}{x}) \xRightarrow{u = \frac{y}{x}} \frac{du}{dx} = u + x\frac{du}{dx} = \varphi(u) \
      \implies x\frac{du}{dx} = \varphi(u) - u \implies \int\frac{du}{\varphi(u) - u} = \int \frac{dx}{x} + C$$

  3. 一阶齐次线性微分方程：

    • 形式：
      $$\frac{dy}{dx} + P(x)y = 0, 各项次数为1$$
    • 解法：
      $$通解公式： y = Ce^{-\int P(x)dx}
      \推导过程如下：
      \1^o:y=0为方程的解
      \2^o:y \not = 0,\frac{dy}{y} = -P(x)dx \implies \ln{\mid y\mid} = -\int P(X)dx + C_0
      \ \implies y= \pm e^{c_0}e^{-\int P(x)dx} \xRightarrow{C = e^{C_0}} y=Ce^{-\int P(x)dx}$$

  4. 一阶非齐线性微分方程：

    • 形式：
      $$\frac{dy}{dx} + P(x)y = Q(x), 等式右边看作0次，即非齐次$$
    • 解法：
      $$常数变易法：令其解为y=c(x)e^{-\int P(x)dx},
      \求导得：\frac{dy}{dx} = c’(x)e^{-\int P(x)dx} - P(x)c(x)e^{-\int P(x)dx}
      \ \therefore c’(x)e^{-\int P(x)dx}=Q(x) \implies c’(x)=Q(x)e^{\int P(x)dx}
      \ \implies c(x) = \int Q(x)e^{\int P(x)dx}dx + C
      \ \therefore y=(\int Q(x)e^{\int P(x)dx}dx + C)e^{-\int P(x)dx}$$

  5. 一阶无特定类的微分方程：

    • 形式1：
      $$\frac{dy}{dx}=\frac{1}{x+y}$$
    • 解法1：
      $$\frac{dy}{dx}=\frac{1}{x+y} \implies \frac{dx}{dy}=x+y \implies x = Ce^y-(y+1)$$
    • 解法2：
      $$令x+y=u,则\frac{dy}{dx} = \frac{du}{dx} - 1,代入：
      \\frac{du}{dx} - 1=\frac{1}{u} \implies u - \ln{(u+1)} = x + C
      \ \implies y-ln(x + y + 1) = C$$
    • 形式2
      $$\frac{dy}{dx}=\frac{1}{(x+y)^2}
      \解法1无效，用解法2即可求解$$

第七章 微分方程（一）

什么是微分方程

  • 方程: 等式 + 未知数

  • 微分方程： 导数/微分 + 方程 —-简称：D.E

例1:曲线y=f(x)过(2,7),每店斜率 = 2x， 求f(x)

解：
$$\frac{dy}{dx} = 2x \quad&&\quad y\mid_{x=2}=7$$
$$\frac{dy}{dx} = 2x \implies y = x^2 + C $$
$$y\mid_{x=2} = 7 \implies C = 3$$
$$y = x^2 + 3$$

微分方程的阶数：

  • 类似方程的次数，微分方程的阶数取决于最高阶项

微分方程的解：

  • 常微分方程：仅允许两个变量存在
  • 微分方程的解为一个函数而非值
  • 存在多个解或无解
  • 分为特解与通解
    • 特解：不含任意常数C
    • 通解：任意常数相互独立且个数与阶数一致
  • 线性无关：
    $$c_1f(x) + c_2y(x) = 0 \iff c_1 = 0 \land c_2=0$$
  • 线性相关：至少一个元素可被其它元素表达成比例
    $$c_1f(x) + c_2y(x) = 0 \implies \exists c_1 \not = 0 \lor \exists c_2 \not = 0$$

微分方程的框架：

  • 三类：
    1. 一阶微分方程，
    2. 可降阶的高阶微分方程
    3. 高阶线性微分方程
